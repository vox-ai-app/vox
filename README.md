# Vox

An open-source AI assistant that lives on your computer — not in the cloud.

It can talk to you, control your screen, manage your files, write documents, send emails, and run tasks in the background. All powered by AI models running locally on your machine. No accounts, no API keys, no data ever leaves your device.

**This project is early.** We're building it in the open and looking for contributors.

## Contributing

```bash
npm install
npm run dev
```

If you're interested in local AI, desktop automation, or open-source tooling — come build with us.
